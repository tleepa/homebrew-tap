# Tleepa Tap

## How do I install these formulae?

`brew install tleepa/tap/<formula>`

Or `brew tap tleepa/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tleepa/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
